# pysignal

A collection of signal processing utilities.

## Getting Started

Clone: `git clone https://github.com/cweave72/pysignal.git`

Build the virtual env (requires `uv`):
```
cd pysignal
. init_venv.sh
```

Run the quick test:
```
(pysignal) python test.py
```

## Running tests

```
(pysignal) pytest -vv pysignal/tests
```
